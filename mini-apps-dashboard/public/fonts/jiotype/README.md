# JioType

WOFF2 files are vendored from Reliance Industries’ public web fonts (`ril.com/themes/custom/reliance/fonts/`).

Weights:

| File                 | CSS weight |
| -------------------- | ---------- |
| `JioType-Light.woff2`  | 300        |
| `JioType-Medium.woff2` | 400, 500   |
| `JioType-Bold.woff2`   | 700        |
| `JioType-Black.woff2`  | 900        |

To refresh: re-download from the URLs above if the CDN updates.
